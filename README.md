# OpenLayout

## Summary

OpenLayout is an open source SwiftUI-inspired layout system. Mostly used for research purposes, but you can use it as you wish!

## Features
- [x] `HStack`/`VStack`
- [x] Fixed `frame`
- [x] Flexible `frame`
- [x] `padding`
- [x] `fixedSize`
- [ ] Text Layout
- [ ] Screenshot Tests
